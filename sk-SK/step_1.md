When we want to represent a colour in a computer program, we can do this by defining the amounts of red, blue, and green that are combined to make up that colour. These amounts are stored as a number between 0 and 255.

![RGB farby](images/RGB.gif)

Nasledujúca tabuľka obsahuje hodnoty niektorých farieb:

| Červená | Zelená | Modrá | Farba     |
| ------- | ------ | ----- | --------- |
| 255     | 0      | 0     | Červená   |
| 0       | 255    | 0     | Zelená    |
| 0       | 0      | 255   | Modrá     |
| 255     | 255    | 0     | Žltá      |
| 255     | 0      | 255   | Purpurová |
| 0       | 255    | 255   | Azúrová   |

Vyberač farieb s ktorým sa dá dobre experimentovať, nájdeš na [webových stránkach w3schools](https://www.w3schools.com/colors/colors_rgb.asp) {:target="_ blank"}.
