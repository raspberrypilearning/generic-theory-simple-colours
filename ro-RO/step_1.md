Când vrem să reprezentăm o culoare într-un program de calculator, putem face acest lucru prin definirea cantităților de roșu, albastru și verde care alcătuiesc acea culoare. Aceste cantități sunt stocate ca un număr între 0 şi 255.

![Culori RGB](images/RGB.gif)

Iată un tabel care prezintă valorile a câteva culori:

| Roșu | Verde | Albastru | Culoare  |
| ---- | ----- | -------- | -------- |
| 255  | 0     | 0        | Roșu     |
| 0    | 255   | 0        | Verde    |
| 0    | 0     | 255      | Albastru |
| 255  | 255   | 0        | Galben   |
| 255  | 0     | 255      | Purpuriu |
| 0    | 255   | 255      | Turcoaz  |

Poți găsi un [selector bun de culoare pe W3Schools](https://www.w3schools.com/colors/colors_rgb.asp){:target="_blank"}.
